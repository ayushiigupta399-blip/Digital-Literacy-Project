##  Project Overview
This project is a Digital Literacy Portfolio created as part of the CSE0001 digital-literacy course. I, being a Student Digital Ambassador at my college, has to help my peers navigate the digital world. The objective is to educate people about digital literacy and to spread awareness on digital communication, online safety and cybersecurity . The main aim of this project is to digitally aware individual, capable of educating others and responsible in using online platforms.
